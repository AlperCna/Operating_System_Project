# ProcX — Process Management System 🐧

**ProcX** is a lightweight **process management system for Linux**, written in **C**. It lets users run and manage programs in the background — similar to `nohup` — while staying accessible across multiple terminal sessions, with all instances kept in sync.

## ✨ Features

- ▶️ Run and manage programs in the background
- 🔄 Control the same processes from multiple terminal sessions
- 🧭 Synchronized state across all connected sessions
- 🐧 Built for Linux environments

## 🛠️ Built With

- **C**
- **Make**

## 🚀 Build & Run

```bash
make
./procx
```
